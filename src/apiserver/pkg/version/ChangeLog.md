# V1.0.0 版本更新日志

### 功能新增

- 支持多生态多版本的 apisix 实例纳管 (APISIX、TAPISIX、BK-APISIX)

- 支持 etcd 数据到控制面编辑区的同步、对比
- 支持一键发布、对比等
- 支持自定义插件管理及配置
